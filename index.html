Changes to CSV

D3 v5 uses Promises instead of asynchronous callbacks for loading data. The use of Promises simplifies the code needed to run the method because it changes the request module to a fetch module. This allows the method to stream the information.

d3 v4 example:
d3.csv("file.csv"), function(error,data){
if (error) console.log(error);
else;
console.log(data);});

d3 v5 example:
d3.csv("file.csv").then(function(data){
console.log(data);
});
